1. 雙擊 `啟動番茄時鐘.bat` 文件
2. 應用會自動在瀏覽器中開啟
3. 開始使用！

## 📱 如何使用 / How to Use

### 番茄時鐘模式 / Timer Modes
- **Focus (專注)**: 25 分鐘工作時間
- **Short Break (短休息)**: 5 分鐘休息
- **Long Break (長休息)**: 15 分鐘休息

### 控制按鈕 / Controls
- **Start (開始)**: 啟動計時器
- **Pause (暫停)**: 暫停計時
- **Reset (重置)**: 重置為當前模式的預設時間

### ⚙️ 通知設定 / Notification Settings
點擊右上角的齒輪圖標 ⚙️ 打開設定面板：

1. **🔇 None (無提示)**: 計時結束時無任何提示
2. **🔔 Brief Sound (簡短聲音)**: 播放簡短的提示音
3. **🎵 Chime (鈴聲)**: 播放較長的鈴聲
4. **💬 Windows Notification + Sound (Windows 通知 + 聲音)**: 
   - 顯示系統通知
   - 同時播放聲音
   - 首次使用需授權瀏覽器通知權限

## 💡 提示 / Tips

- 使用 **Focus** 模式進行 25 分鐘的深度工作
- 完成後使用 **Short Break** 放鬆 5 分鐘
- 每完成 4 個番茄時鐘後，使用 **Long Break** 休息 15 分鐘
- 根據個人喜好選擇通知方式
- 保持瀏覽器標籤頁開啟以確保計時器正常運行

## ❓ 常見問題 / FAQ

**Q: 計時器在後台會繼續運行嗎？**  
A: 是的，只要瀏覽器標籤頁保持開啟，計時器就會繼續運行。

**Q: 我可以自定義時間嗎？**  
A: 目前預設為標準番茄時鐘時間（25/5/15 分鐘）。

**Q: Windows 通知沒有顯示？**  
A: 請確保已授權瀏覽器通知權限，並且系統通知功能已開啟。
